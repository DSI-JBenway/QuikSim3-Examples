# Stress Relaxation

Ramp a specimen to 20% strain and hold the displacement for 300 seconds while
recording force decay. Useful for characterising viscoplastic behaviour and
calibrating constitutive models.

## What it demonstrates

- Slow strain-rate tensile deformation (0.1 s⁻¹)
- Extended post-deformation dwell (hold)
- Low acquisition rate appropriate for slow relaxation events

## Key parameters

| Parameter        | Value      |
|------------------|------------|
| Peak temperature | 800 °C     |
| Soak time        | 30 s       |
| Strain           | 0.2 (eng)  |
| Strain rate      | 0.1 s⁻¹    |
| Dwell (hold)     | 300 s      |
| Acquisition rate | 100 Hz     |

## How to use

Open in QuikSim3, save a copy via **File → Save As** into your project, then
adjust the dwell or strain to suit your material.
